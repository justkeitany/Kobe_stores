# IPTV Panel

Personal IPTV panel with Xtream Codes API, HLS streaming, and admin dashboard.

## Install

See `install.sh` — run as root on Ubuntu 22.04.

## Update

```bash
sudo bash /opt/iptv-panel/scripts/update.sh
```

## Reset password

```bash
sudo bash /opt/iptv-panel/scripts/reset-password.sh
```
